# Public Voice

This dashboard should read like a sharp operating board, not a startup landing page and not a software spec.

## Voice rules

- direct
- grounded
- useful
- confident without sounding inflated
- real details over generic claims
- motion over fluff

## Avoid

- placeholder filler
- vague productivity language
- hypey founder-speak
- overexplaining the system to death

## Aim

The site should feel like Bryon has real things in motion and a clean way to see them.
